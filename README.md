# Node.JS Weather App

Study project.

"The Complete Node.js Developer Course (2nd Edition) " (Udemy)

## Install

````bash
npm install
````

## Usage

Get temperature:
````bash
node app.js -a 'yekaterinburg'
````
or
````bash
node app-promise.js -a 'yekaterinburg'
````

Help:
````bash
node app.js -h
````
